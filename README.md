# Website
This is a simple web project using HTML, CSS, and JavaScript.This project demonstrates key Human-Computer Interaction (HCI) principles using a simple interactive web interface.
It visually highlights concepts like affordance, feedback, mapping, constraints, and signifiers through user interaction.

📌 Features

🔵 Affordance
The button has a blue border indicating it is clickable.

🟢 Mapping
The status box clearly shows the result of user actions.

🟡 Feedback
When the button is clicked:
It changes color (yellow)
Slight animation (scale effect) is applied

🔴 Constraint
After 3 clicks:
Button gets disabled
Red border indicates restriction

🟣 Signifier
Disabled button changes appearance (purple + faded look)

⚙️ How It Works

➜ User clicks the button.

➜ Click count increases.

➜ Visual feedback is applied (color + animation).

➜ Status message updates dynamically.

➜ After 3 clicks:

➜ Button is disabled

➜ UI visually communicates restriction
